    compose_createEOBSolutions

    @mixin compose_createEOBSolutions($compositePages, $solutionPage, $sectionHeaderNode) { ... }

    Description

    Create solutions at the end of the book (by creating a new collated Page). These can be separated by section, chapter, both, or none.

    When they are separated by chapter/section a new element is created with the Chapter/Section title.

    Additionally, if sectionSeparated is selected for the EOB composite page, the originating EOC section must also contain sectionSeparated.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $compositePages

    each element contains source and hasSolutions

    		List—none
    $solutionPage

    contains source, sectionSeparated, and chapterSeparated

    		Map—none
    $sectionHeaderNode—noneString—none

    Requires

    • [mixin] utils_checkType
    • [mixin] utils_checkTypeOptional

    utils_title

    @mixin utils_title($content, $bucket: null, $container: null, $containerType: null) { ... }

    Description

    Base titling and numbering mixin - Creates elements before an element defined by the content passed. These elements can be automatically wrapped and/or sent to a specified bucket

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $content

    A content map containing key and value pairs. Each key, value pair creates a of class="key" with a text node of "value"

    		Map—none
    $bucket

    The unquoted name of the bucket to send the resulting object(s) to. If null, resulting object(s) will remain in place, before the element the mixin was called within

    		Textnull
    $container

    The unquoted type of container element to send the titling spans to (e.g. h1, h2, div). If null, resulting spans will not be wrapped

    		Textnull
    $containerType

    The data-type to give to the created container element. If null, resulting spans will not be wrapped

    		Stringnull

    Used by

    • [mixin] compose_titleEOCComposites
    • [mixin] compose_titleEOBComposites
